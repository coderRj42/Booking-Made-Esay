TO INSTALL XAMPP:
.XAMPP CAN BE INSTALLED THROUGH THE LINK:https://www.apachefriends.org/download.html
.VISUAL STUDIO CODE CAN BE DOWNLOADED THROUGH:https://code.visualstudio.com/download


TO RUN THE PROGRAM:
.EXTRACT THE FILE
.CREATE A FOLDER AND PLACE THE FILE IN XAMPP->HTDOCS
.IN XAMPP,START THE APACHE AND MYSQL SERVER
.OPEN PHPMYADMIN AND IMPORT THE DATABASE FILE
.EXECUTE THE FILE IN SERVER
.FORMAT:localhost/filename/homepage.html
